Session Trad Audomaroise

Présentation

L'idée de cette Session Trad ou Folk Audomaroise est de pouvoir se réunir dans le secteur de Saint-Omer pour pratiquer tous types de répertoires de musiques traditionnelles. 

Dans la configuration actuelle, la session est une "slow session" incluant des morceaux entendus principalement en bal folk du Nord et du Pas de Calais : morceaux de Flandre, de France, d'Irlande, d'Europe...
